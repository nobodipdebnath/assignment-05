Can you explain what the Document Object Model (DOM) is and its core functions?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree-like model, allowing developers to manipulate its content and visual presentation. The DOM’s core functions include creating, reading, updating, and deleting nodes. Nodes represent elements, attributes, and text within the document. By manipulating these nodes, developers can dynamically change content and styles. For instance, JavaScript often uses the DOM API to interact with HTML elements in web development.

How would you manipulate the DOM using JavaScript?

JavaScript manipulates the DOM by selecting elements and changing their properties or contents. The ‘document’ object is used to access the HTML document, with methods like getElementById() or querySelector() for selection. Once an element is selected, its attributes can be changed using dot notation. For example, if we have a paragraph with id “demo”, we could change its text content as follows:

let para = document.getElementById("demo"); para.textContent = "New Text";

How do you select individual elements in DOM using JavaScript?

In JavaScript, individual elements in the DOM can be selected using various methods. The most common are ‘getElementById’, ‘getElementsByClassName’, and ‘getElementsByTagName’.
